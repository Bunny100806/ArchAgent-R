## 🌟 Overview

ArchAgent-R is an AI-powered software architecture reasoning platform that transforms natural language project requirements into complete software architecture designs.

The platform leverages **Llama 3.2**, **FastAPI**, **React**, **SQLite**, and **React Flow** to provide an end-to-end architecture generation environment.

### ✨ Key Capabilities

✅ Requirements Analysis

✅ Architecture Pattern Selection

✅ System Decomposition

✅ Architecture Validation

✅ Interactive Architecture Canvas

✅ AI Architecture Assistant Chat

✅ PNG & PDF Export

✅ SQLite Project History

✅ Secure User Authentication

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Bunny100806/ArchAgent-R.git
cd ArchAgent-R
```

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

### Ollama

```bash
ollama pull llama3.2
ollama serve
```
